About Me

---this is me---

Hi, I'm Schalk.

I'm in love with web development, and I am passionate about web solutions connecting users with their interests and each other. I am fortunate enough to call the Bushveldt my home. More particularly a small town called Modimolle, better known as Nylstroom. It is situated in the Limpopo Province, the northern part of South Africa, about a 120km drive from Tshwane, Pretoria in the Gauteng Province. Some might say, a chip and a putt from where I stay.

When I'm not developing, you will find me somewhere outside, enjoing time with my family. I'm a family man and love to have fun my wife and daugther. As we love the Bushveldt, we are naturally drawn to the beauty of nature. Short hikes in the veldt is just one of our activities we love doing together.

I love to read and play guitar. Otherwise, I love to cook and braai! BBQ's are for sissies!

I enjoy both front-end and back-end web development. My passion is to solve problems and implement solutions that is meaningful and creative! I enjoy engaging clients for constructive feedback to solve problems and finding solutions together. I like to develop what users see in their browser or application as it gives me an opportunity to be creative. I also enjoy using logic to serve up sites and applications which render a great user interface and experience! I focus on responsive design and maintainable code that scales easily, and improves web performance!

The server side of an application, how it works and everything that communicates between the database and the browser is what interests me most. I like to get my head under the hood to boost engine performance for a smooth ride and don't mind getting my hands dirty. Web pages shouldn't be boring, but rather be dynamic, responsive, fast and intuitive with mobile first in mind! Easy to use user interfaces and better experiences allow the user to focus on content.

My Skills

---what I'm skilled at---

User Experiance

I design dynamic, responsive, fast and intuitive web applications and pages. The user interface and experience I incoporate promotes functionality and it is a top priority! I like to keep the audience captivated! I like the user to complete the task on the website or application they set out to do easily and without confusion. Ultimately, is not just about buttons and menus, but the whole user experience.

Fast

I use light weight designs to create fast loading web pages, with lag-free interaction, which increases visitor retention and loyalty!

Responsive

I design responsive web pages, with mobile first in mind, that render well on a variety of devices and window or screen sizes.

Intuitive

My pages rely on easy to use user interfaces and better user experiences that allow the user to focus on the product or service!

Dynamic

I like to use active pages which comes to life. It keeps the user curious and captures their interest.
